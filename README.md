# Checklist Rural

Sistema local em Go para atendimento, triagem, checklist e controle de pendências em projetos rurais.

## Objetivo

Este sistema foi criado para auxiliar no atendimento inicial de produtores rurais, organizando:

- Cadastro da reunião
- Dados do produtor
- Banco pretendido
- Tipo de projeto rural
- Triagem inicial
- Checklist automático
- Controle de itens pendentes, recebidos ou não aplicáveis
- Resumo para WhatsApp
- Exportação TXT
- Backup local do banco SQLite

## Tecnologias usadas

- Go
- HTML
- CSS
- SQLite
- modernc.org/sqlite
